Типы селекторов

*

Универсальный селектор - иногда требуется установить одновременно один стиль для всех элементов веб-страницы (использую вкачестве установки шрифта)

А

В качестве селектора может выступать любой элемент HTML (применил к h2)

#id

Идентификатор определяет уникальное имя элемента

.class

Классы применяют, когда необходимо определить стиль для одного или нескольких элементов веб-страницы.

Мультиклассы

В таком случае стиль применяется только для элементов, у которых одновременно заданы классы class1 и class2

A B

Вложенные селекторы - можно одновременно установить стиль для отдельного элемента, а также для элемента, который находится внутри другого.

A>B

Дочерние селекторы - Дочерним называется элемент, который непосредственно располагается внутри родительского элемента.

A+B

Соседние селекторы - cоседними называются элементы веб-страницы,когда они следуют непосредственно друг за другом в коде документа.

A~B

Pодственные селекторы-по своему поведению похожи

поведению похожи на соседние селекторы

(запись вида E + F) но в

отличие от них стилевые

применяются ко всем близлежащим элементам.
